ConcealJS

customize your password input field

View project on GitHub

Hide passwords in plain sight

Use this Ultra simple library to change any input into a funky password field

start by initializing concealer object

var fancyInput = new concealer();

the conceal the desired input field

fancyInput.conceal('.myinput','myfunkypasswordmarkup');

and whenever you want to unhide or take a peek at your password

fancyInput.reveal();

var password = facnyInput.reveal();

once done taking a peek you can re hide your password with your funky markup line

fancyInput.reconceal();

and remember to include the script on your html
